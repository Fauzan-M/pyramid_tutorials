=====================================
Basic Backend with Python and Pyramid
=====================================

Let's use Chapter 3 of the Pyramid Quick Tutorial, but converted into
a Nix build environment

Steps
=====

#. ``cd basic_backend``

#. ``python setup.py develop``

#. ``pserve development.ini``
